JSON转换Model方法

2016-01-07

常用第三方库Json转Model

我目前用的比较多的库MJExtension,YYModel,JSONModel

MJExtension

安装方式

cocoapods：在Podfile中添加pod ‘MJExtension’,添加建议使用pod update (此处两个-)no-repo-update,不然速度奇慢

手工：下载文件导入主头文件 #import “MJExtension”

使用

导入主头文件 #import “MJExtension”

YYModel

安装

CocoaPods

pod 'YYModel'

pod install 或 pod update。
