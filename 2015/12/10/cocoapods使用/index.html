cocoapods使用

2015-12-10

CocoaPods

写在前面，我的环境是OS X 10.11.1 Xcode 7.1

CocoaPods是第三方依赖库管理工具, 在iOS开发中非常流行，基本是成为第三方依赖库管理的事实标准。

官网

github地址

如何使用

1.修改ruby源,官方是国外的源，由于某种神奇的东西，速度奇慢，还经常上不了，改为国内的淘宝源

2.安装cocoapods

3.cd 工程的根目录

切换到工程根目录

4.touch Podfile

创建Podfile文件

5.vim Podfile(写入内容)

pod ‘xxx’,’~> y’

xxx表示第三方库名称，y表示第三方库的版本号
6.在Podfile中添加需要的库

7.pod search XX

搜索名字XX的库，搜索结果会显示如何添加到Podfile中
8.pod install

9.工程名字.xcworkspace

install 结束后，根目录下会多出个工程名字.xcworkspace的文件，双击打开工程就行了

10.在build setting 添加searchpath（$SRCROOT）

会在#import时提醒

11.不升级CocoaPods的spec仓库

pod update –verbose
